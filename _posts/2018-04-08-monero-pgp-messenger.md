## Install

- install [IPFS](https://ipfs.io/docs/install/)
- pip3 install flask, ipfsapi, blueprint, jsonify, requests
- clone repo && cd monero-pgp-messenger/app
- python3 app.py
- in web browser, go to: 127.0.0.1:5000
